# CHEM 6634 
Quantum Chemistry and Spectroscopy: Online materials

### Installation
1. Download
    
        git clone https://github.com/nmayhall-vt/CHEM6634.git
        cd CHEM6634/

2. create virtual environment
         
        virtualenv -p python3 venv
        source venv/bin/activate

3. Install

        pip install .
